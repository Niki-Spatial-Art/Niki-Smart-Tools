# Action Card Rules

## Intraday Defaults

- If the user asks "要不要买", default to no new risk unless a clear pullback setup exists.
- If the user already owns the theme, prefer hold/no add.
- If the candidate is a hot stock after a sharp intraday rise, say "加自选，不追".
- If the candidate was bought today, remind T+1 and avoid adding unless pre-planned.
- If the trade is meant to "补亏", reject revenge trading and reframe as risk control.

## Position Sizing

Use the user's latest account context first. If unknown:

- ETF trial: usually 5,000-10,000 RMB.
- Single stock trial: usually 1 board lot or 5,000-10,000 RMB, whichever is smaller and allowed by lot size.
- Same-theme add: smaller than the first position.
- Do not use bank cash as chase capital.

## Sell/Clear Priority

Clear or reduce in this order:

1. Old weak positions unrelated to current main themes.
2. Positions that failed their original thesis.
3. Same-theme duplicates bought from FOMO.
4. Winners only when trend breaks, valuation/position is uncomfortable, or the account needs risk reduction.

## Morning/Afternoon Handling

- Morning low-open pullbacks can be valid if index support holds.
- Afternoon chase after a strong rebound is usually not a buy point.
- Before holidays, reduce new risk unless the setup is exceptional.
- If index support breaks, stop adding and only handle weak positions.
