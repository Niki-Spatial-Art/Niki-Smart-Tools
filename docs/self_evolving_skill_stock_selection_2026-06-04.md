# 华泰自进化 Skill 选股：系统实验清单

生成时间：2026-06-04
来源：用户粘贴的华泰金工“自进化 Skill—选股策略的自动迭代”相关资料。
定位：策略研发流水线，不是一个可以直接实盘买入的信号。

## 我们怎么理解这里的 Skill

这里的 `Skill` 不是“今天买哪只股票”的黑箱，而是一个可复用研究模块：

```text
市场假设
  -> 因子定义
  -> 数据拉取
  -> 回测验证
  -> 错误审计
  -> 参数/规则迭代
  -> 动作卡表达
```

用户是研究总监，AI 是分析师。AI 负责实现和检查，用户负责交易假设、风险边界和是否执行。

## 最小可落地版本

先不做复杂自动进化，先做“半自动迭代”：

| 阶段 | 输入 | 输出 |
| --- | --- | --- |
| 1. 假设 | 一句话策略逻辑 | `hypothesis` |
| 2. 因子 | iFind 可取字段 | 可计算信号 |
| 3. 回测 | 历史行情 | T+1/T+2/T+5 胜率和收益 |
| 4. 审计 | 前视偏差、样本数、追高、幸存者偏差 | 审计日志 |
| 5. 动作卡 | 通过/不通过 | 买/等/不买规则 |
| 6. 复盘 | 真实成交 | 更新策略样本 |

## 第一批适合做的实验

| 实验 | 假设 | 数据 | 通过标准 |
| --- | --- | --- | --- |
| 回踩承接 | 强势股回踩不破 MA20 后更容易延续 | iFind 日线/实时 | 样本 >= 20，T+2 中位收益 > 0，胜率 > 55% |
| 放量不过热 | 量能 5/20 抬升但涨幅未过热时更安全 | iFind 成交额/涨幅 | 追高样本剔除后仍有效 |
| 主题共振 | 同层级多股走强比单股走强更可靠 | watchlist 层级 + iFind | 同层级强势数提升后胜率改善 |
| 止损线有效性 | 跌破预设线后继续持有是否劣化 | 真实交易 + 历史行情 | 证明止损不是情绪动作 |

## 自动迭代边界

允许自动化：

- 生成候选因子；
- 写回测脚本；
- 跑参数网格；
- 检查样本数和前视偏差；
- 输出实验报告。

禁止自动化：

- 自动下单；
- 为了提高回测收益反复调参直到过拟合；
- 软止损后继续生成进攻买入；
- 把“策略迭代结果”直接变成实盘指令。

## 接入路线

1. 先升级 `ifind_position_backtest.py`，让每个动作卡带样本、胜率、中位收益、过热过滤。
2. 再新增 `tools/factor_experiment_runner.py`，专门跑小型因子实验。
3. 每个实验输出到 `reports/factor_experiments/`。
4. 只有连续复盘有效的规则，才回写动作卡模板。

## 对当前交易的影响

今天不因为这个 Skill 开新仓。它提升的是未来研究效率和动作卡质量，不改变当前风控闸门。
