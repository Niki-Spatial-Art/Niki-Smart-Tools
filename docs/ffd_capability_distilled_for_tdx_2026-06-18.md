# FFD 能力蒸馏到通达信工作台

生成日期：2026-06-18

## 结论

FFD 不再作为当前工作台的 MCP 接入项。原因是 Codex 侧持续遇到 `api.wanzhuan.cc/responses 503`，继续围绕 MCP 排查会消耗注意力。

但 FFD 的产品能力有参考价值，可以蒸馏成通达信工作台的功能蓝图。保留“它启发了什么”，不保留“必须接它”。

当前主线：

1. 通达信作为盘中主工作台。
2. DeepSeek / 问小达作为公式、复盘、逻辑解释助手。
3. 本地公式筛选器负责过滤源码指标。
4. FFD 只作为能力参考，不作为盘中依赖。

## FFD 能力拆解

| FFD 能力 | 原用途 | 通达信工作台吸收方式 | 当前动作 |
|---|---|---|---|
| 全球新闻 | 找事件、催化、宏观冲击 | 做“盘前/盘中事件清单”模板，来源可用通达信资讯、财联社、同花顺、网页 | 不接 MCP，保留模板 |
| 研报资料库 | 搜行业报告、公司报告 | 做“研报观点 -> 板块/个股/验证项”拆解模板 | 暂用网页或手工贴入 |
| A 股行情 | 查历史/实时行情 | 通达信直接看行情；公式和选股在通达信内执行 | 通达信优先 |
| 公告 | 查上市公司公告 | 通达信 F10/公告 + 手工摘要，重点看定增、减持、业绩、解禁、回购 | 做公告检查清单 |
| 财务指标 | ROE、利润同比、营收同比、PE/PB | 通达信 F10 + 条件选股过滤，后续可做本地财务表 | 先人工验证 |
| 行业/产业指标 | 跟踪行业景气，如 PCB、铜、锂、油价 | 转成“板块景气观察卡”，不直接下单 | 保留框架 |
| 资金流向 | 主力、北向、板块资金 | 通达信资金流向、盘口、量价验证 | 只作辅助，不作单因子买点 |
| 选股/筛选 | 按条件找股票 | 通达信条件选股 + 本地公式筛选器 | 主力方向 |
| 技术指标/风险指标 | RSI、MACD、回撤、波动 | 通达信公式 + 本地回测/观察记录 | 主力方向 |
| 新闻情绪/复合投研 | 把新闻、资金、研报串起来 | 做“动作卡”：题材强度、前排、失效条件、仓位 | 手工/半自动 |

## 要复制的是结构，不是数据源

FFD 给工作台的启发是五层：

1. 情报层：今天发生了什么。
2. 映射层：影响哪些行业、概念、公司。
3. 行情层：市场是否确认这个方向。
4. 公式层：能否转成可执行的通达信条件。
5. 动作层：买、等、加、减、放弃，以及失效条件。

## 公式方向

FFD 的技术指标能力，迁移到通达信里就是三件事：

1. 批量筛源码公式：先淘汰未来函数、重绘、L2 依赖。
2. 把指标公式改成 `XG:` 选股公式。
3. 对每个选股公式建立观察记录，而不是直接实盘。

## 暂停项

- FFD MCP 接入。
- 围绕 503 继续排查。
- 把 FFD API 做成盘中必需数据源。
- 把新闻/研报输出直接变成买入指令。
