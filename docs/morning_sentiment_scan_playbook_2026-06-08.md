# 盘前舆情情绪扫描规则

生成日期：2026-06-08

用途：把海外 X、国内社区和新闻快讯先整理成“题材与风险映射”，再交给 iFind、回测和风控闸门验证。舆情扫描不能直接生成买入指令。

## 每日扫描时间

- 默认接入 `每日 9:20 盘前动作卡`。
- 只在 A 股交易日输出盘前摘要。
- 如 X、集思录、社区或快讯源无法访问，必须写明“不可验证”，不能编造。

使用规则：

- 海外源只做题材、供应链和风险映射。
- 不从海外 X 直接生成 A 股买点。
- 若海外科技链承压，A 股 PCB、CPO、半导体、算力硬件默认降低追高等级。
- 若海外 AI 基建长期逻辑增强，只进入观察池或研究层，仍需 iFind 验证成交额、量比、MA20 距离和回测样本。

## 国内社区情绪

优先观察：

- `集思录`：ETF、QDII、可转债、A/H 比价、期权、低风险资金偏好。
- `淘股吧`：短线主线、连板、亏钱效应、情绪退潮或抱团。
- `雪球`：中长逻辑、产业链解释、机构叙事。
- `东方财富股吧`：只当情绪温度计，不当交易依据。

使用规则：

- 社区观点不能直接变成交易指令。
- 集思录重点看低风险品种是否出现信用风险、溢价拥挤或套利退潮。
- 淘股吧重点看主线是否扩散，还是只有少数抱团票。
- 雪球和股吧只帮助发现叙事，不能替代 iFind 数据。

## 国内快讯与政策

每天盘前整理与持仓、观察池或大盘风控有关的快讯：

- 政策与监管：定增、再融资、产业政策、财政和发改委方向。
- 港股科技：恒生科技、互联网平台、港股半导体。
- 半导体 / CPO / PCB / 被动元件 / MLCC。
- 机器人 / 减速器 / 工程机械 / 空间计算。
- 贵金属、锂电、固态电池、工业金属。
- 地缘、航运、汇率、海外利率和信用风险。
- 灾害、防汛、农业、区域基础设施等事件。

快讯输出只保留能影响以下三类判断的内容：

1. 持仓是否需要降风险。
2. 观察池是否需要升级或降级。
3. 当天是否适合新开仓。

## iFind 验证字段

所有舆情结论必须回到 iFind 或本地回测验证：

- 最新价和涨跌幅。
- 成交额。
- 量能 5/20。
- MA20 距离。
- 5 日 / 20 日涨幅。
- T+1 / T+2 / T+5 回测样本。
- 持仓止损线、处理线、仓位占比。
- 当日亏损软硬闸门。

未通过验证的结论只能写成：

- `观察`
- `不做`
- `等回踩`
- `需要二次确认`

不能写成：

- `买入`
- `加仓`
- `追`
- `确定性机会`

## 盘前输出格式

盘前动作卡里增加一段“舆情温度”：

```text
## 舆情温度

海外映射：
- 1-3 条最重要的海外风险或题材。

国内社区：
- 1-3 条低风险/短线情绪变化。

快讯政策：
- 1-3 条能映射到持仓或观察池的快讯。

对今天的影响：
- 持仓风险：提高 / 不变 / 降低
- 新开仓闸门：允许 / 只观察 / 禁止
- 需要 iFind 验证：列出具体字段
```

## 硬规则

- 社区和新闻负责发现风向，iFind 负责验证强弱，动作卡负责管住手。
- 强势观察不等于买入候选。
- 没有 fresh iFind 数据时，不输出高置信买入动作卡。
- 当日亏损接近或超过硬止损线时，舆情利好只能进入观察池，不能开新仓。
- 旧仓风险未处理完之前，新增题材默认降级。
- 任何真实交易都必须由使用者手动确认和执行。
