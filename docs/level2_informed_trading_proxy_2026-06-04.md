# Level2 知情交易因子：无 Level2 阶段替代方案

生成时间：2026-06-04
来源：用户粘贴的广发金工 Level2 知情交易因子资料。
定位：高价值研究方向，但当前系统未接入 Level2，因此先做代理版。

## 价值判断

真正的知情交易因子通常需要：

- 委托队列；
- 主动买卖方向；
- 大单/超大单；
- 撤单行为；
- 逐笔成交；
- 盘口深度。

当前系统以 iFind HTTP 和本地券商截图为主，不应伪装成拥有 Level2。

## 可先做的代理指标

| 代理指标 | 数据源 | 用途 |
| --- | --- | --- |
| 量比 | iFind 实时 | 判断资金活跃度 |
| 成交额 5/20 | iFind 历史 | 判断放量是否持续 |
| 当日涨跌幅 | iFind 实时 | 过滤追高 |
| MA20 距离 | iFind 历史 | 判断是否过热/回踩 |
| 20日回撤 | iFind 历史 | 判断高位风险 |
| 同层级强势数 | watchlist + iFind | 判断主题共振 |

## 盘中动作卡用法

没有 Level2 前，不能写“知情资金流入”。只能写：

- 量能放大；
- 成交额相对 20 日均值抬升；
- 同主题多股共振；
- 回踩承接或高位过热。

## 禁止事项

- 不把普通成交额包装成 Level2 大单因子。
- 不用单一量比给买入指令。
- 不在亏损软止损后用盘口强势票追回。

## 后续接入条件

如果以后接入 Level2 或更细粒度盘口数据，再新增：

- 主动买入占比；
- 大单净流入稳定性；
- 撤单/挂单异常；
- 分钟级冲击成本；
- 开盘 15 分钟知情交易强度。

这些指标必须先做离线回测，再进入动作卡。
