# 45万本金月入6万目标工作流

更新时间：2026-06-19  
用途：把「上周表现复盘 + 回测模拟 + 市场热度预测 + 下周买卖预案」固化为工作台提问流程。  
边界：研究与纪律工具，不自动交易，不承诺收益。

## 目标换算

| 口径 | 数字 |
|---|---:|
| 总本金 | 450,000 |
| 月目标收益 | 60,000 |
| 需要月收益率 | 13.33% |
| 粗略周目标 | 15,000/周 |
| 粗略日目标 | 3,000/交易日 |
| 若只用证券账户约30.6万 | 需要约19.6%月收益率 |

结论：这是进攻型目标，不适合用「每天必须赚」的方式硬做。工作台应优先跟踪：

- 是否遵守计划
- 是否只做主线和双跌买点
- 单笔最大亏损是否可控
- 是否有 20-50 笔样本证明正期望
- 是否避免用补亏心态追高

## 每周流程

### 1. 上周表现复盘

输入：

- 上周持仓截图或券商导出的成交记录
- 星耀行情与 K 线
- monitor.py 持仓雷达
- 梅森交易知识库主题逻辑

输出：

| 模块 | 要回答的问题 |
|---|---|
| 账户收益 | 上周总盈亏、日内最大回撤、现金比例 |
| 交易归因 | 哪些收益来自市场 beta、主题选择、个股选择、择时 |
| 纪律检查 | 哪些交易是计划内，哪些是 FOMO、追高、补亏 |
| 持仓质量 | 核心仓、试错仓、拖累仓、可清理仓 |
| 样本更新 | 这周哪些 setup 值得继续，哪些需要淘汰 |

### 2. 回测与模拟

不要只回测「买了会赚多少」，要同时回测「不买会怎样」。

必须做三类模拟：

1. **实际路径**：按真实买卖复盘。
2. **纪律路径**：如果只买双跌买点、追高不买，会怎样。
3. **风险路径**：如果每笔亏损到失效线就出，会少亏还是错过反弹。

输出字段：

```text
标的
真实买入/卖出
纪律买入/卖出
最大浮亏
最大浮盈
是否追高
是否重复同方向
如果重来，下一次规则怎么改
```

### 3. 市场热度与下周主线

用星耀和 monitor.py 生成热度，再用梅森资料库解释。

优先看：

- 涨跌家数
- 成交额变化
- 量比
- 行业/概念强弱
- ETF 强弱
- 持仓相对强弱
- 是否接近均线偏离过高
- 是否存在监管/节假日/外部事件风险

主题分层：

| 等级 | 含义 | 动作 |
|---|---|---|
| A | 主线强、逻辑强、价格未过热 | 可等双跌买点 |
| B | 逻辑强但短线过热 | 只观察不追 |
| C | 逻辑一般但有修复 | 小仓试错或不做 |
| D | 弱仓/旧仓/拖累 | 减或清 |

### 4. 下周买卖预案

每周最多给：

- 3 个核心持仓动作
- 3 个候选观察
- 3 个明确不做

每个标的必须输出：

```text
动作：买 / 不买 / 持有 / 减 / 清
触发：价格/指数/板块条件
失效：跌破哪里或什么逻辑证伪
仓位：金额/份额/股数上限
风险：T+1、重复主题、追高、节假日
```

## 月入6万的现实约束

### 不允许的做法

- 为了凑 6 万目标而每天强行交易
- 把微众银行现金转入证券账户追涨
- 同一主题重复买 3-5 个标的
- 因为群里热闹就追涨
- 把亏损票无限等反弹

### 允许的做法

- 主线明确时，提高持仓质量
- 双跌买点小仓试错
- 只在正期望 setup 上逐步放大
- 错了快速减小风险
- 每周只优化 1-2 条规则

## 一键综合提问

```text
用梅森交易知识库 + 星耀实时数据 + monitor.py 持仓雷达，基于我上周表现做周复盘、回测模拟和下周预案。

我的终极目标是45万本金月收益6万，但请先判断这个目标在当前市场下是否应该进攻、保守推进，还是暂停放大。

要求输出：
1. 上周收益归因：市场beta、主题选择、个股选择、择时、仓位。
2. 纪律复盘：哪些符合梅森双跌买点，哪些是追高/FOMO/补亏。
3. 回测模拟：真实路径 vs 纪律路径 vs 风险路径。
4. 当前市场热度：用星耀看行业、ETF、成交额、量比、涨跌家数。
5. 下周关注：最多3个核心持仓动作、3个候选观察、3个明确不做。
6. 每个标的给动作、触发价、失效线、仓位上限。
7. 月入6万目标进度：本周需要赚多少，最大允许亏多少，是否允许加大仓位。
8. 不要建议我动用微众银行现金追涨。
```

## 盘前版提问

```text
基于上周复盘和今天盘前星耀数据，给我今天的执行计划。
只输出：
核心持仓怎么处理、候选票是否等待双跌买点、今天禁止做什么、单票最大金额。
```

## 盘后版提问

```text
基于今天成交和持仓截图，更新月入6万目标进度。
判断今天是纪律加分还是扣分。
如果今天亏了，明天是防守、修复还是继续进攻？
```
