# 因子研究与动作卡验证模板

生成时间：2026-06-04
用途：把用户盘中直觉、研报主题、技术指标转化为可回测、可复盘、可进入动作卡的因子实验。

## 一句话原则

任何“我觉得会涨”的想法，必须先变成可计算因子，再进入动作卡。

## 因子实验模板

| 字段 | 填写要求 | 示例 |
| --- | --- | --- |
| 直觉假设 | 一句话说明为什么可能有效 | 放量回踩后更容易次日延续 |
| 股票池 | 明确范围 | 持仓 + 强势观察 + 主题池 |
| 数据源 | iFind / 本地 CSV / Level2 / 公开数据 | iFind 日线 + 实时行情 |
| 因子定义 | 可重复计算 | 5日均量 / 20日均量，收盘相对 MA20 |
| 信号形态 | 连续值/0-1/分档 | 量能 5/20 > 1.3 且价格在 MA20 上 |
| 交易方向 | 买入/卖出/减仓/禁买 | 买入候选或观察降级 |
| 验证窗口 | T+1/T+2/T+5 | 次日、2日、5日收益 |
| 风控条件 | 止损、仓位、日亏损闸门 | 软止损触发后禁买 |
| 通过标准 | 样本数、胜率、收益、回撤 | 样本 >= 20，T+2 中位数 > 0，胜率 > 55% |
| 动作卡输出 | 买/等/不买 | 等回踩买，不追高 |

## 最小验证四件套

| 验证项 | 看什么 | 适合判断 |
| --- | --- | --- |
| IC/RankIC | 连续因子和未来收益相关性 | 因子排序是否有用 |
| 分类胜率/Precision | 0/1 信号命中后是否更容易上涨 | 事件型短线信号 |
| 事件研究 | 信号后 T+1/T+2/T+5 的收益分布 | 动作卡持有周期 |
| 冗余检查 | 与已有因子是否重复 | 避免同一信号包装多次 |

## 进入买入卡的最低门槛

一只票进入“买入候选卡”前，必须同时满足：

- 账户风控允许新买；
- 不是今日已卖出禁止买回标的；
- 当日涨幅未进入追高禁区；
- iFind 数据新鲜；
- 至少有一个可解释因子支持；
- 回测样本没有明显反向；
- 有明确止损和仓位上限。

不满足以上条件时，只能进入观察卡。

## 盘中使用方法

```text
主题/直觉出现
  -> 写成因子定义
  -> 查 iFind 当前状态
  -> 查相似历史样本
  -> 过风控闸门
  -> 输出买/等/不买
```

## 禁止事项

- 不用研报标题直接生成买点。
- 不用“强势观察”替代买入候选。
- 不在日亏损软止损后扩大风险。
- 不把单次回测胜率当成稳定收益承诺。
